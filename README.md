## 프로젝트 개요
- S&P 500 종목들에 대해 데이터를 수집 합니다.
- 수집한 데이터를 통해 기본적/기술적 지표로 가공합니다.
- 지표를 기반으로 기계학습을 수행하고, 모델을 통해 주가 변동에 대해 예측을 수행합니다.

## 주요 기능
- SQLite 기반 주가 및 재무 데이터 파이프라인 구축
- Awesome Oscillator, KAMA 등 40가지 이상의 퀀트 투자 지표 계산
- 계산된 지표와 주가 데이터 기반으로 Random Forest, XGBoost, LSTM 등의 기계학습을 통한 주가 방향 예측

## 기술 스택
- **언어**: `python`
- **환경**: `WSL2 Ubuntu`, `Miniforge3`
- **기계학습**: `cuML(GPU 가속)`, `scikit-learn`, `XGBoost`
- **DB**: `SQLite`

## 한계점
중단된 프로젝트지만, 한계점에 대해 한번 고찰해 보았습니다.
### 타겟 변수 설계
- 주가 변동의 시점과 양이 제각각이라 적절한 타겟 변수를 설정하는데 어려움이 있었습니다.
- **개선안**: 고정된 미래 시점의 주가 및 변동률 보다는 n일 내 상승/하락 Threshold에 도달했는지를 타겟으로 사용
### 무분별한 지표 사용
- 지표에 대한 이해가 깊지 않아, 학습 데이터의 적절성이 부족했다고 생각합니다.
### 모델 성능
- 상기의 문제점들로 인해 만족할만한 모델 성능이 나오지 않았다고 생각합니다.
### 백테스트 부재
- 만족할만한 예측률이 나오지 않아 백테스트까지 진행하지 않았습니다.
